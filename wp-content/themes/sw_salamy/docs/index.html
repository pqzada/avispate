This theme is a fixed layout with two columns. All of the information within the main content area is nested within a div with an id of "primaryContent". The sidebar's (column #2) content is within a div with an id of "secondaryContent".

How To Edit Color

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

#primaryContent a {
  color: #someColor;
}

Change Structure

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

#wrap #primaryContent a {
  color: #someColor;
}

Styles Styles customization and highlights

Heads up! I'm using two CSS files in this theme. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

Installation How to Install the Template

1. Install Theme

Here's a brief information about how to install your WordPress theme from scratch

1.1) Install WordPress
  • Sign up for web hosting if you do not already have hosting (recommended hosts).
  • Make a new installation of the latest version of WordPress on your hosting
    • account. If you're new to all this, see Installing WordPress and New to WordPress. Some hosts have an auto-install feature.
  • If you will be replacing a WordPress site, make a complete backup first.
1.2) Upload and Activate Theme
  • Log into your WordPress dashboard and navigate to Appearance > Themes > Install Themes > Upload.
  • Choose the zip file you downloaded then click install. If you receive an error saying the file is too big.
  • After installation, click Activate.
1.3) Installing simple content
To install the Sample Content
  • Go Dashboard > Tools > Import > WordPress
  • Install the plugin.
  • Use it to upload the sample content XML file .

Shortcodes

Shortcodes make it simple to insert elements like buttons, boxes, columns etc. without any knowledge how to code HTML. Shortcodes work in posts and pages .

Icon

Use Font Awesome, the iconic font designed for Bootstrap. Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.

Get the code: [icon name="icon-glass" type="icon-4x" tag="i" color="red" ]

  • name: Name of icon or icon class, reference at http://fontawesome.io/icons/. Accept “ICONNAME” without prefix “icon-”.
  • type: icon-4x/icon-3x/icon-2x
  • tag: html tag name to render icon
  • color: color of icon, if not set, default text color of site style will effected.

Button

The [button] shortcode can be used to make small and large buttons of various colors.

Get the code: [button id="" class="btn" type="primary" target="" href="#" tag="span"]link[/button]

  • type: size and color of button; large, small, mini for sizing; primary, danger, warning, success, info, inverse for color.
  • tag:html tag name to render button
  • target và href: Use when tag="a"
Size

Utilize the extra button types type="large", type="small", or type="mini " for sizing.

Alert

Alerts Styles for success, warning, and error messages

Get the code: [alert class="alert" type="success" dismiss="true" tag="div"][/alert]

- type: block/success/error/infomation.

- dismiss: true/false

Default alert [alert class="alert" type="block" dismiss="true" tag="div"]Wrap any text and an optional dismiss button in .alert for a basic warning alert message.[/alert]

Dismiss buttons [alert class="alert" type="block" dismiss="false" tag="div"]Wrap any text and an optional dismiss button in .alert for a basic warning alert message.[/alert]

Google maps

You can embed a custom Google Map using the [googlemaps] shortcode.

Get the code: [googlemaps title="Envato Office" location="Hoan Kiem District, Hanoi, Vietnam" zoom="10" Width="100%" height=350]

Slideshow

With the built-in, you can insert awesome image everywhere, just use the Code provided when Adding a Slideshow.

  • Get the code: [slideshow ids="" caption="" size="" interval="" event="" class=""]
  • ids: id of attachment.
  • caption: true/false.
  • size: thumbnail/medium/large.
  • interval: The amount of time to delay between automatically cycling an item. If false, carousel will not automatically cycle.
  • event: slide/slid.
  • class: You can custom with your class you put in.

    - Tiny grid (<768 px) : .col-xs-*

    - Small grid (>768 px) : .col-sm-*

    - Medium grid (>992 px) : .col-md-*

    - Large grid (>1200 px) : .col-lg-*

    * : Number of column(s) ( from 1 - 12 )

Collapse

Get base styles and flexible support for collapsible components like accordions and navigation.

    Get the code:
    [collapses class="collapses" tag="div"]
      [collapse title="Section 1"]Accordion Content 1[/collapse]
      [collapse title="Section 2"]Accordion Content 2[/collapse]
      [collapse title="Section 3"]Accordion Content 3[/collapse]
    [/collapses]
  • class: as “name” above function.
  • tag: html tag name to render icon.
  • title: as name of each above content. [collapse]

Tabs

    Wrap multiple [tab] shortcodes with the [tabs] shortcode.

    Get the code:
    [tabs possition="top" class="tabs" tag="div"]
      [tab title="Section 1"]Tab Content 1[/tab]
      [tab title="Section 2"]Tab Content 2[/tab]
      [tab title="Section 3"]Tab Content 3[/tab]
    [/tabs]
  • position: top/right/bottom/left
  • class: as “name” above function.
  • tag: html tag name to render icon.
  • title: title of tab

Columns

    Display content such as columns reponsive.

    Get the code:
    [row tag="div"]
      [col class="" tag="div" large="" medium="" small="" xsmall=""]Content[/col]
      [col class="" tag="div" large="" medium="" small="" xsmall=""]Content1[/col]
    ...
    [/row]
  • class: as “name” above function.
  • tag: html tag name.
  • large : Number of column(s) in large grid ( > 1200px ).
  • medium : Number of column(s) in medium grid ( > 992px ).
  • small : Number of column(s) in small grid ( > 768px ).
  • xsmall : Number of column(s) in tiny grid ( < 768px ).
  • Number of columns : From 1 to 12.

Pricing Table

    This template contains pricing shortcode. You just select shortcode to display.

    Get the code:
    [pricing_table style="style2"]
      [pricing style="style2" featured="no" description="Instant access, all resources." plan="Premium Plus" cost="$75.00" per="/mo" button_url="#"
    button_text="Sign up now" button_target="self" button_rel="nofollow"]Content[/pricing]
      [pricing style="style2" featured="no" description="Business standard." featured="yes" plan="Professional" cost="$45.00" per="/mo" button_url="#"
    button_text="Sign up now" button_target="self" button_rel="nofollow"]Content[/pricing]
      [pricing style="style2" featured="no" description="Just give it a try." plan="Basic(free)" cost="Freemium" per="" button_url="#"
    button_text="Sign up now" button_target="self" button_rel="nofollow"]Content[/pricing]
    [/pricing_table]
  • style : have 2 styles: style1 (3 columns) and style2(5 columns).
  • featured : have 2 value: yes or no.
  • description : Description of pricing column.
  • plan : Title of pricing column.
  • cost : Price of pricing column.
  • per : Price per day, week, month or year.
  • button_url : Url of the button.
  • button_text : Title of buttom.
  • button_target : The target attribute specifies where to open the linked document (self, blank, parent, top)
  • size : When style is style1, size will has value: one-five.
  • button_color : When style is style1, you can choose color of pricing button.

Tooltip

    Shortcode show tooltip with content and postition of tooltip.

    Get the code:
    [ya_tooltip content="content" orient="top"]Title[/ya_tooltip]
  • - content: Content of the tooltip.
  • - orient: Position of the tooltip(left, right, top, bottom).

Modal

    Get the code:
    [ya_modal label="title" header="header" close="close" save="save"]Content[/ya_modal]
  • - Label: label of button launch.
  • - Header: Title will show in modal header.
  • - Close: Label of button close.
  • - Save: Label of button save.

Recent post slider

    Show recent post as slider

    Get the code:
    [ya-recent-posts limit="6" title="Recent posts" num_column="3" id="1"]
  • -Limit: the number of posts to show.
  • -Title: The title of header slider.
  • -Num_column: the number of columns to show.
  • -Id: The ID to show many slider recent posts (each slider have one ID).

Menus

Support 2 custom menu:

  • Dropdown menu

  • Mega menu

    Support to enhance the Wordpress default menu system, with a variety of useful features to edit menus and give theme a unique look.
    The advanced options allow the user to gain more control over the design of menus. The user can make several columns in menus, divide those columns and place their alignment as left and right as required by your website

    1. Creat menu

      YA Framework support for new Wordpress menu system, to manage your main menu items, please bellow step.
      Go to Apperance -> Menus, first you need to creat menu place for Main Navigation, for more detail please follow this image:

    2. Manage and ordering your menu items

      Description

      1. Icon:

        Using icon-* class to display Icon for any menu item with Font Awesome intergrated. Refercence Font Awesome for details

      2. Dropdown Size:

        - Only for menu items level 1
        - Validate values are span1, span2, …, span12. These are fully compatible with Bootstrap Style.
        - If ‘None’ is selected, default dropdown-menu (as Dropdown) will be display.
        - If this item has no child menu item, this config have no efftected for anything.

      3. Dropdown Align:

        - Only for menu items level 1
        - If you want dropdown (if have) align by right direction, config it to ‘Right’ instead of ‘Left’ or ‘Default’
        - ‘Default’: no class be add to dropdown, everything are nature by default css.
        - ‘Left’ / ‘Right’: add to dropdown an utility class for float it left/ right. Make it align as you want.

      4. Show Description as Subtitle

        - Mean as the name.
        - Description is core attribute of menu item. You may need check to show it in the metabox screen.

      5. Hide Title

        Sometime you need hide the title of menu, select ‘Yes’ value please. In this case, Icon or Subtitle or Advanced Content are need to show as replacer.

      6. Advanced

        - Some advanced functions are placed here.
        - ‘Default’: NOOP, do nothing. Sure, it is not advanced
        - ‘Use as Divider’: Show any icon, title, subtitle, content. Divider style only.
        - ‘Append Advanced Content’: Append content after menu title block (icon+title+subtitle)
        - ‘Append Advanced Content with Shortcode’: as above, but ‘do_shortcode’ with this before append.

      7. Advanced content

        Insert content follow Advanced.

      8. Span

        - Only for menu items level 2
        - Only effect if parent item have validate size (span1-span12).
        - Important: This should not be greater parent dropdown-size
